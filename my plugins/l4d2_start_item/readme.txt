    left4dhooks https://forums.alliedmods.net/showthread.php?t=321696

功能
    在第一位玩家离开安全区域后，按配置文件设置给予玩家与Bot物品，用于解决多人模式下医疗物资不够用、三方图不刷初始枪械时人机变笨或者铸币出门忘带基础物资的情况
    配合删除安全区域物资插件使用更佳哦

指令与Cvars
    l4d2_start_item_slot0   控制给予玩家的主武器种类
    l4d2_start_item_slot1   控制给予玩家与Bot的副武器种类(目前并未开放)
    l4d2_start_item_slot2   控制给予玩家与Bot的投掷物种类
    l4d2_start_item_slot3   控制给予玩家与Bot的医疗物资种类
    l4d2_start_item_slot4   控制给予玩家与Bot的小药种类
    l4d2_start_item_botweapon   控制给予Bot的主武器种类
    l4d2_start_item_bgetrule    控制给予Slot0-4的规则
    l4d2_start_item_pgetrule    控制给予Bot主武器的规则
    l4d2_start_item_allowplaymusic  控制出门是否播放音效
